The DataHawk API helps you manage tracked products, keywords, categories, tags, and integrations programmatically.

## Important identifiers

- **API key** in **Settings** > **API Reference**
- **Workspace ID** for debugging and support

## Typical use cases

- Track and untrack entities
- Check marketplace integrations
- Manage tags and source state
