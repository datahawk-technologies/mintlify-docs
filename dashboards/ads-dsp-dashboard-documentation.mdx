---
title: "Ads DSP Dashboard Documentation"
---

The DSP dashboard provides Amazon DSP campaign analysis across impressions, clicks, spend, detail page views, sales, and new-to-brand metrics.

## Requirements

Connect an Amazon DSP account.

## Main sections

- DSP Ads Report
- Metrics Definition

## Main filters

- Date and granularity
- Advertiser
- Country
- Currency
- Order and line item
- Creative filters
- Optional custom parameter filters
